The Beginning for customizing Windows 11

You can customize an ISO of Windows 11 with multiples ways.

I recommend the way 1, because it's the best and this way is easy

Way 1: Customizing an ISO with NTLite

You have just to download NTLite from this Official Mirror

If at any point the software prompts you for a license, select Community (free) and then click OK

Then, modify the ISO as desired following the instructions provided by the software.

Way 2: Customizing an ISO with MSMG Toolkit

You have just to download the MSMG Toolkit from this Official Mirror

Extract the Software if is it a .zip file or a .rar file.

Then, modify the ISO as desired following the instructions provided by the software.
